**IT Hacks**

### Top Time-Saving Windows Shortcuts
These are some of the most useful shortcuts you can use on Windows to save time while working:
*   `Ctrl + Shift + Z`: This shortcut allows you to undo multiple actions without having to click the undo button again.
*   `Alt + Tab`: You can quickly switch between open windows using this shortcut.
*   `F1` or `Windows key + R`: You can access various Windows settings and tools by pressing these keys. For example, you can type "time" in the search bar to view system time.

### Keyboard shortcuts for Mac
These are some of the most useful shortcuts you can use on a Mac to save time while working:
*   `Cmd + Z`: This shortcut allows you to undo multiple actions without having to click the undo button again.
*   `Cmd + C` or `Cmd + V`: These shortcuts allow you to copy and paste text in applications like Mail, TextEdit, and Pages.
*   `Cmd + Shift + >` or `Cmd + Shift + <<`: You can quickly increase and decrease font sizes using these shortcuts.

### Quick Fixes for Everyday Tech Headaches
Here are some quick fixes you can use on Windows and Mac to resolve common tech-related issues:
*   **Slow PC**: Try closing unnecessary programs, updating your operating system, and cleaning up disk space.
*   **No internet connection**: Check your network settings, restart your router or modem, and try connecting to a different Wi-Fi network.
