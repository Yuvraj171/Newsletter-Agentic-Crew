This Month’s Spotlight: Microsoft Planner

Microsoft Planner is your team's new best friend for staying organized and on top of tasks. Think of it as a digital whiteboard where you can list projects, assign tasks, set deadlines, and track progress – all in one place. It's designed to be simple and visual, so everyone on the team can easily see what needs to be done and who's doing it. And with the new Microsoft Planner unifying Planner, Project for the web, and To Do, it's more powerful than ever!

Why This Matters

For BEST Group, Planner can make a real difference in how we manage production, collaborate on projects, and keep everything running smoothly. It helps reduce confusion, avoid missed deadlines, and ensure everyone is on the same page. By using Planner, we can:

  • Improve Production Planning: Easily break down production processes into manageable tasks.
  • Boost Team Collaboration: Provide a central place for communication, file sharing, and progress updates.
  • Increase Efficiency: Streamline workflows and reduce the time spent on administrative tasks.
  • Reduce Errors: Ensure tasks are clearly defined and assigned, minimizing mistakes and rework.

How You Can Use Microsoft Planner at BEST Group

Planner can be used across various departments at BEST Group to streamline workflows and improve collaboration:

  • HR: Manage employee onboarding, training schedules, and performance reviews.
  • Finance: Track budgets, manage expense approvals, and prepare for audits.
  • Legal/Compliance: Manage regulatory compliance tasks, review contracts, and update policies.
  • IT: Track help desk tickets, manage software deployments, and plan infrastructure upgrades.
  • Sales & Marketing: Coordinate marketing campaigns, track sales leads, and manage customer feedback.
  • Production: Track the progress of manufacturing tasks, assign responsibilities, and manage deadlines.

Smart Features You Should Be Using

Feature Name What it Does Why You'll Love It How to Do It
Task Prioritization Allows you to set the priority of tasks (urgent, important, medium, low). Ensures the most critical tasks are addressed first, improving efficiency. When creating or editing a task, select the desired priority level from the dropdown menu.
Bucket Organization Organizes tasks into different categories or stages (e.g., To Do, In Progress). Provides a clear visual representation of project progress. Create new buckets to categorize tasks and drag and drop tasks between buckets to update their status.
Charts and Dashboards Provides visual representations of project progress and team workload. Offers a quick overview of project health and helps identify potential issues. Navigate to the 'Charts' tab within a plan to view automatically generated charts and dashboards.
Teams Integration Access and manage Planner plans directly within Microsoft Teams channels. Streamlines communication and collaboration by keeping project info in one place. Add the Planner app to a Teams channel to create or access existing plans.

Pro Tips to Try This Week

  • Start with a Template: Don't start from scratch! Explore the available templates to quickly set up a plan for common projects.
  • Use Labels: Create custom labels to categorize tasks further (e.g., "Quality Control," "Shipping").
  • Set Realistic Deadlines: Be honest about how long tasks will take to avoid unnecessary stress and delays.
  • @Mention Team Members: Use @mentions in task comments to notify specific people and keep them in the loop.
  • Check the "My Tasks" View: See all your assigned tasks across different plans in one convenient location.
