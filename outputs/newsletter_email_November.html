IT Hacks

These simple IT hacks will save you time and frustration!

Top Time-Saving Windows Shortcuts

  • Ctrl + C: Copy
  • Ctrl + V: Paste
  • Ctrl + X: Cut
  • Ctrl + Z: Undo
  • Ctrl + A: Select All
  • Alt + Tab: Switch Windows
  • Windows Key + D: Show Desktop

Top Time-Saving Mac Shortcuts

  • Cmd + C: Copy
  • Cmd + V: Paste
  • Cmd + X: Cut
  • Cmd + Z: Undo
  • Cmd + A: Select All
  • Cmd + Tab: Switch Applications
  • Cmd + Space: Open Spotlight Search

Quick Fixes for Everyday Tech Headaches

Slow Wi-Fi

  • Restart your computer and router.
  • Check your Wi-Fi signal strength.

Printer Not Connecting

  • Check the printer cable.
  • Restart the printer.
  • Update the printer drivers.

Application Frozen

Windows: Press Ctrl+Shift+Esc to open Task Manager, then end the frozen application.

Mac: Press Cmd+Option+Esc to open the Force Quit window, then select and quit the frozen application.

Computer Running Slow

  • Close programs you aren't using.
  • Windows: Run Disk Cleanup and defragment your hard drive (if it's not an SSD).
  • Mac: Run Disk Utility to repair disk permissions and check Activity Monitor for resource-heavy processes.

Web Browser Not Responding

  • Close tabs that aren't responding.
  • Clear your browser's cache and cookies.
  • Restart the browser.

No Sound

  • Check the volume levels.
  • Make sure the correct audio output is selected.
  • Windows: Restart the audio service.
  • Mac: Restart Core Audio.

Pro-Tips

  • Restart your computer weekly to clear temporary files and boost performance.
  • Use a password manager (like LastPass or 1Password) for strong, unique passwords.
  • Clear your browser's cache and cookies regularly for better speed and privacy.
  • Back up your important files to an external drive or cloud storage to prevent data loss.
