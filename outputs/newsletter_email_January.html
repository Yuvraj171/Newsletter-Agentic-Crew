IT Hacks

Save time and boost your productivity with these simple IT hacks!

Top Time-Saving Windows Shortcuts

  • Ctrl + C: Copy
  • Ctrl + V: Paste
  • Ctrl + A: Select All
  • Alt + Tab: Switch Windows
  • Windows Key + D: Show/Hide Desktop
  • Windows Key + L: Lock PC
  • Ctrl + Shift + Esc: Open Task Manager

Top Time-Saving Mac Shortcuts

  • Cmd + C: Copy
  • Cmd + V: Paste
  • Cmd + A: Select All
  • Cmd + Tab: Switch Apps
  • Cmd + Spacebar: Spotlight Search
  • Cmd + W: Close Window
  • Cmd + Option + Esc: Force Quit

Quick Fixes for Everyday Tech Headaches

Slow Wi-Fi

  • Windows: Restart router and computer. Check Wi-Fi signal. Update network drivers.
  • Mac: Restart router and computer. Check Wi-Fi signal. Renew DHCP Lease (System Preferences > Network > Wi-Fi > Advanced > TCP/IP).

Printer Not Connecting

  • Windows: Check cable. Restart printer and computer. Update drivers.
  • Mac: Check cable. Restart printer and computer. Re-add printer (System Preferences > Printers & Scanners).

Application Frozen

  • Windows: Press Ctrl+Shift+Esc, select app, and click 'End Task'.
  • Mac: Press Cmd+Option+Esc, select app, and click 'Force Quit'.

Computer Running Slow

  • Windows: Close unused programs. Run Disk Cleanup. Defragment HDD (if not SSD).
  • Mac: Close unused programs. Run Disk Utility to repair permissions. Check Activity Monitor.

Web Browser Not Responding

  • Windows: Close tabs or restart browser. Clear cache and cookies.
  • Mac: Close tabs or restart browser. Clear cache and cookies.

Pro-Tips

  • Restart your computer weekly.
  • Use a password manager.
  • Clear browser cache and cookies regularly.
  • Take breaks from your screen hourly.
