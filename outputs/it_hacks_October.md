# Top Time-Saving Windows Shortcuts  
- **Print Screen (PrtSc)**: Captures a screenshot of your entire screen.<br>  
  Use this when you need quick images of layouts or backgrounds.  

- **Ctrl + C and Ctrl + V**: Copies text from one location to another, like the clipboard to Notepad.<br>  
  Faster than clicking the Clipboard button.  

- **Win + Settings**: Opens Windows settings for system management.<br>  
  Access essential controls with a single key combination.  

---

# Keyboard shortcuts for Mac  
- **Finder (Cmd + F)**: Navigates between windows in Finder.<br>  
  Efficiently switch tabs without using the mouse.  

- **Cmd + C and D**: Closes a window by clicking another one. Drag to close multiple windows at once.<br>  
  Removes multiple windows with just two keys.  

- **Cmd + Shift + F1**: Opens System Preferences for settings like enlighten and time zone.<br>  
  Reveals vital system options not found in Finder.

# Quick Fixes for Everyday Tech Headaches  
- **Slow Wi-Fi connection**:
  - Windows: Press `Win + Settings`, go to Network & Internet, select Turn off adaptive bitrate.  
  - Mac: Use `Command + Option + Z` (Adaptive bitrate) or `Command + W` (Static bitrate).  

- **Printer not connecting**:
  - Windows: Open Print Management Center in Control Panel.<br>  
    Go to devices and ensure the printer is listed.  
  - Mac: Select the printer from Finder and click Get Ready.  

- **Application frozen**:
  - Windows: Press `Ctrl + Shift + Esc` to end the application immediately.  
  - Mac: Use `Command + Control + Option + E` to close applications quickly.  

- **Computer running slow**:
  - Windows: Type `Ctrl + Shift + Task Manager` and restart with `Esc`.  
  - Mac: Run `Command + Control + Option + R`, type "taskmgr", press Enter, then Command + D for a forced restart.
